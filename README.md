# Simple Image Gallery

This is a simple image gallery website built using React. The project displays a collection of images in a gallery format.

## Features

- Display a collection of images in a gallery layout.
- Responsive design for mobile and desktop views.
- Simple and clean user interface.

## Installation

To get a local copy up and running follow these simple steps:

1. **Clone the repository:**

    ```sh
    git clone https://github.com/your-username/simple-image-gallery.git
    ```

2. **Navigate to the project directory:**

    ```sh
    cd simple-image-gallery
    ```

3. **Install the dependencies:**

    ```sh
    npm install
    ```

## Usage

1. **Start the development server:**

    ```sh
    npm start
    ```

2. **Open your browser and visit:**

    ```
    http://localhost:3000
    ```

3. **View and interact with the image gallery.**
